# Version Control Practice Repo

This repository is created for **Introduction to Software Engineering & Computing Lab**.  
The purpose is to **practice version control systems** (Git & GitHub), including making commits, creating and merging branches, and testing pull requests.

## How to Contribute

1. Clone the repository:
```

git clone [https://github.com/eyustarX/Practice-project.git](https://github.com/eyustarX/Practice-project.git)

```
2. Create a new branch:
```

git checkout -b your-branch-name

```
3. Make changes to files
4. Stage and commit:
```

git add .
git commit -m "Describe your changes"

```
5. Push your branch:
```

git push origin your-branch-name

```
6. Open a **Pull Request** on GitHub for review

> This repo is for lab practice only.
